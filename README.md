# Wordle Game (Python Tkinter)

## Overview

This is a Wordle-style desktop game built using Python and Tkinter. The player has 6 attempts to guess a randomly selected 5-letter word. After each guess, the game provides feedback using colors:

* 🟩 Green: Correct letter in the correct position
* 🟨 Yellow: Correct letter in the wrong position
* ⬜ Gray: Letter not in the word

## Features

* Interactive GUI built with Tkinter
* Automatic cursor movement between input boxes
* Input validation (only letters, one character per box)
* Real-time feedback with color-coded results
* Random word generation from a predefined list
* Game reset functionality
* Win/Loss detection

## Technologies Used

* Python
* Tkinter (GUI Framework)

## How to Run

1. Make sure Python is installed (Python 3.x recommended)
2. Clone this repository:

   ```bash
   git clone https://github.com/yourusername/wordle-game.git
   ```
3. Navigate to the project folder:

   ```bash
   cd wordle-game
   ```
4. Run the program:

   ```bash
   python main.py
   ```

## How to Play

* Enter a 5-letter word using the input boxes
* Press Enter or click Submit
* Use the color feedback to guide your next guess
* You have 6 attempts to guess the correct word
